# Modern Portfolio - Mohammed Arshaq

A sleek, modern, and highly aesthetic portfolio website built with a focus on immersive user experience, premium glassmorphism design, and smooth scroll animations.

## 🚀 Built With

- **HTML5** - Semantic structure and content.
- **CSS3** - Custom properties, modern Grid/Flexbox layouts, and glassmorphism styling.
- **JavaScript (ES6+)** - Dynamic typing effects, intersection observer for scroll-reveal animations, and high-resolution image modal logic.
- **FontAwesome** - Professional iconography.
- **Google Fonts (Outfit)** - Premium typography.

## ✨ Key Features

- **Glassmorphism Design**: Frosted glass effects and subtle glowing borders throughout for a premium look.
- **Interactive Hero Section**: Animated background blobs and a custom typing/erasing effect for roles.
- **Experience & Education Timelines**: Professional tracking of career and academic milestones with branded logos.
- **Interactive Tech Stack**: Categorical pill-based skills section with hover-glow interactions.
- **Image Lightbox**: High-resolution certificate viewing built directly into the Certifications section.
- **Reveal on Scroll**: Seamless fade-in and slide-up animations as you explore different sections.
- **Fully Responsive**: Optimized for every screen size, from desktops to mobile devices.

## 📂 Project Structure

- `index.html` - The main entry point and content of the site.
- `css/style.css` - Custom design system, typography, and animation definitions.
- `js/script.js` - Dynamic interactions and scroll observation engine.
- `assets/images/` - Repository for project screenshots, logos, and certificates.
